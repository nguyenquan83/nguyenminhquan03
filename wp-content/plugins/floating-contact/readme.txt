Integrates a floating contact button and opens an modal contact form.

== Description ==
The „Floating Contact Button“ displayes an fixed contact button on the bottom right of the screen. It opens an modal with your favorite contact form plugin. Before you have to generate a contact form and after this copy the shortcode into the plugin settings form Floating Contact Button.

The plugin is real lightweight. The .css file is minimized to load as fast as can. 

== Installation ==
First download the the plugin ZIP file.

1.) Log into your website administration panel.
2.) Then go to Plugins page Add New > Upload Plugin .
3.) Click „Choose file“ then select the plugin .zip file.
4.) Then Install and activate the plugin.

Plugin Settings

After the plugin is activated, you can find in the menu the „Floating Contact“ settings. Here you insert your contact form shortcode. This will displayed into the contact form modal. Here it is also possible to change the basic color of the button and to disable the rotation on hover.

== Frequently Asked Questions ==
= Which contact form plugin work with the Floating Contact Button? =
Its tested with Contact Form 7, Ninja Forms, Caldera Forms & HappyForms. The requirement is an shortcode.

= Can i change the style itself? =
In the plugin settings you can only change the basic color for the button. To change the elements in the modal you have to make custom css.

= Can i change the style from the insert plugin shortcode? =
Optical adjustments of the used contact plugin have to be done user-defined e.g. in the theme own style.css.

= Where can I report questions or problems with the plugin? =
Please use the plugin own support forum.

= Are there any known themes with which the plugin does not work without errors? =
Unfortunately yes. The plugin does not work with the McLuhan theme.
